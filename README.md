# 📝 Todo App

A simple and responsive **Todo App** built with **HTML**, **Bootstrap**, and **Vanilla JavaScript**.

## 📋 Features

- ✅ Add new tasks
- ✅ Mark tasks as completed
- ✅ Delete tasks
- ✅ Responsive layout using Bootstrap 5

## 🛠️ Technologies Used

- HTML5
- Bootstrap 5
- JavaScript (Vanilla)
